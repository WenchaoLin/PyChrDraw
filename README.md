# PyChrDraw: a python library for chromosome drawing

## Settings yaml file

```yaml
# List of chromosome names
chr_names:
  - A1
  - A2
  - A3
# Dict of chromosome sizes
chr_sizes:
  A1: 239302903
# Dict of chromosome centromere positions
chr_cen_pos:
  A1: 85059547
# Dictionary with band information
# (A1, 13020, 1500, '#CCCCCC')
bands:
  A1:
    - (13020, 1500, '#CCCCCC')
```
